1.3 齊次常微分方程式

Homogeneous Ordinary Differential Equations

第一部分：核心概念

🤔 課程摘要

並非所有的一階 ODE 都能直接分離變數。然而，有一類特殊的方程式，稱為「齊次微分方程式」，可以透過巧妙的變數代換，將其轉換為可分離變數的形式來求解。本章節的核心就是學習如何辨識齊次 ODE 並掌握這種代換技巧。

  • 齊次函數 (Homogeneous Function)
    一個函數 $f(x, y)$ 如果滿足 $f(tx, ty) = t^n f(x, y)$，則稱其為 $n$ 次齊次函數。簡單來說，就是將 $x$ 換成 $tx$、$y$ 換成 $ty$ 後，可以提出一個 $t^n$ 的公因數。
  • 齊次微分方程式 (Homogeneous ODE)
    對於全微分形式 $M(x,y)dx + N(x,y)dy = 0$，如果 $M(x,y)$ 和 $N(x,y)$ 都是同次數的齊次函數，則此方程式為齊次 ODE。另一種等價的判斷方式是，若方程式可寫成 $y' = f(\frac{y}{x})$ 的形式，它就是齊次的。
  • 關鍵的變數代換
    解齊次 ODE 的關鍵技巧是進行變數代換。我們有兩種選擇：
    1. 令 $y = ux$，則 $dy = u dx + x du$。
    2. 令 $x = vy$，則 $dx = v dy + y dv$。
    通常，如果 $N(x,y)$ 比 $M(x,y)$ 簡單，我們會選擇 $y=ux$；反之，若 $M$ 比 $N$ 簡單，則選 $x=vy$ 會讓計算稍微簡便。
  • 解題標準流程
    1. 檢查齊次性: 確認 $M, N$ 為同次齊次函數。
    2. 變數代換: 選擇 $y=ux$ 或 $x=vy$ 並計算 $dy$ 或 $dx$。
    3. 代入化簡: 將代換式代入原 ODE，化簡並消去共同的 $x^n$ 或 $y^n$ 項。
    4. 分離變數: 化簡後的方程式會變成一個關於新變數 (u 或 v) 和原自變數 (x 或 y) 的可分離變數 ODE。
    5. 積分求解: 對分離變數後的方程式兩邊積分。
    6. 還原變數: 將 $u=\frac{y}{x}$ 或 $v=\frac{x}{y}$ 代換回來，得到原方程式的解。
